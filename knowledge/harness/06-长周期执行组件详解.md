# Harness 组件详解 - 长周期自主执行（Long Horizon Execution）

> **组件定位**: 完成复杂、长期、自主的任务
> **重要性**: ⭐⭐⭐⭐⭐（智能体的终极能力）
> **实现难度**: ⭐⭐⭐⭐⭐（需要完整系统）

---

## 🎯 核心价值

### 三大挑战

```
1. 过早停止
   └─ 做了一半就说"我完成了"

2. 分解困难
   └─ 不知道怎么拆分复杂任务

3. 连贯性差
   └─ 换了个上下文就忘了前面在做什么
```

---

## 🏗️ 三大解决方案

### 1. 文件系统 + Git 追踪

**状态持久化**：
```python
# 第 1 轮（上下文 A）
write_file("feature.py", "登录功能代码")
execute("git commit -m '完成登录'")

# 第 2 轮（上下文 B）
log = execute("git log")  # 看到已完成登录
write_file("feature.py", "注册功能代码")
execute("git commit -m '完成注册'")

# 第 3 轮（上下文 C）
log = execute("git log")  # 看到已完成登录、注册
write_file("feature.py", "首页功能代码")
```

### 2. Ralph 循环（强制继续）

**原理**：
```python
def ralph_loop(task: str, max_iterations: int = 10):
    """Ralph 循环"""
    for i in range(max_iterations):
        # 执行任务
        result = agent.execute(task)

        # 检查是否完成
        if is_complete(result):
            break

        # 拦截退出，重新注入
        task = f"""
        原始任务：{task}

        当前状态：
        {read_file("plan.md")}

        继续执行，不要停止！
        """

        # 在新上下文中继续
        context = create_new_context()
```

### 3. 规划 + 自验证

**规划文件**（plan.md）：
```markdown
# 任务：开发 Web 应用

## 步骤
- [x] 1. 设计数据库
- [x] 2. 实现登录功能
- [ ] 3. 实现注册功能
- [ ] 4. 实现首页
- [ ] 5. 部署上线

## 当前进度
已完成登录功能，正在开发注册功能
```

**自验证循环**：
```python
def self_verification_loop():
    """自验证循环"""
    while not all_tasks_complete():
        # 完成一步
        task = get_next_task()
        result = execute_task(task)

        # 运行测试
        test_result = run_tests()

        if test_result.failed:
            # 修复错误
            fix_errors(test_result.errors)
        else:
            # 更新进度
            update_plan(task, status="完成")
```

```python
def decompose_complex_task(task: str) -> List[str]:
    """分解复杂任务"""
    # 使用 LLM 分解
    prompt = f"""
    任务：{task}

    请将这个任务分解为具体的步骤：
    1.
    2.
    3.
    """
    steps = llm.generate(prompt)
    return parse_steps(steps)
```

### 2. 进度追踪

```python
class ProgressTracker:
    """进度追踪器"""

    def track(self, step: str, status: str):
        """追踪进度"""
        log = f"[{datetime.now()}] {step}: {status}\n"
        append_file("progress.log", log)
```

```python
def recover_from_error(error: Exception):
```
