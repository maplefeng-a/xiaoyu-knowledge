# Harness 组件详解 - 记忆和搜索（Memory & Search）

> **组件定位**: 持续学习 + 实时知识
> **重要性**: ⭐⭐⭐⭐（智能体的记忆）
> **实现难度**: ⭐⭐⭐（需要检索技术）

---

## 🎯 核心价值

### 两大核心能力

```
1. 记忆（Memory）
   └─ 跨会话持久化知识

2. 实时知识（Real-time Knowledge）
   └─ 突破知识截止日期限制
```

---

## 🤔 为什么需要记忆和搜索？

### 问题 1：模型记不住

```
第 1 天：
用户：我叫小明，喜欢 Python
智能体：好的，记住了

第 2 天：
用户：我喜欢什么语言？
智能体：我不知道（上下文已清空）
```

### 问题 2：知识有截止日期

```
用户：今天 OpenAI 发布了什么？
智能体：我的知识截止到 2025年1月
       不知道今天发生的事
```

---

## 🏗️ 实现原理

### 记忆系统架构

```
启动时：
MEMORY.md → 注入上下文 → 模型"回忆"

工作中：
学到新知识 → 写入 MEMORY.md → 持久化

下次启动：
读取 MEMORY.md → 注入上下文 → 继续学习
```

### 搜索系统架构

```
用户提问
   ↓
判断是否需要实时信息
   ↓
调用 web_search
   ↓
搜索最新内容
   ↓
返回结果
```

---

## 💻 实际案例

### 案例 1：MEMORY.md 系统

```markdown
# MEMORY.md

## 用户偏好
- 名字：小明
- 喜欢的语言：Python
- 工作领域：数据分析

## 项目信息
- 当前项目：销售数据分析
- 技术栈：pandas, matplotlib

## 学习记录
- 2026-03-10：学习了数据可视化
- 2026-03-11：掌握了时间序列分析
```

### 案例 2：实时知识获取

```python
# 智能体获取最新信息
query = "OpenAI 最新发布 2026-03-11"
results = web_search(query)

for result in results:
    print(f"标题：{result.title}")
    print(f"摘要：{result.snippet}")
```

```markdown
- 重要任务列表
```

### 2. 定期更新

```python
def update_memory(new_info: str):
    """更新记忆文件"""
    memory = read_file("MEMORY.md")
    updated = memory + f"\n\n## 新增（{date}）\n{new_info}"
    write_file("MEMORY.md", updated)
```
