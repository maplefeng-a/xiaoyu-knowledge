# Msg 消息模型

> agentscope-java 基础设施层

## 一句话

不可变的多模态消息抽象，支持文本、图片、音频、工具调用等多种内容类型。

## 核心设计

- **不可变性**: 所有字段 final，Builder 模式
- **多模态**: List<ContentBlock> content 支持多种类型
- **角色**: USER/ASSISTANT/SYSTEM/OBSERVER

## ContentBlock 类型

- `TextBlock` - 文本
- `ImageBlock` - 图片
- `AudioBlock` - 音频
- `ToolUseBlock` - 工具调用
- `ToolResultBlock` - 工具结果
- `ThinkingBlock` - 思考过程
