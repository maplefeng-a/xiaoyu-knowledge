# 多智能体协作

> agentscope-java 集成分析

## 一句话

MsgHub 消息总线 + Pipeline 工作流，实现多智能体协作编排。

## 核心组件

### MsgHub
- 发布-订阅模式
- 类型安全消息路由
- 支持一对一、一对多、多对多通信

### Pipeline
- SequentialPipeline - 顺序执行
- FanoutPipeline - 并行执行
- Pipeline 链式组合

## 协作模式

```
┌─────────┐    ┌─────────┐    ┌─────────┐
│ Agent A │───▶│ MsgHub  │───▶│ Agent B │
└─────────┘    └─────────┘    └─────────┘
                    │
                    ▼
              ┌─────────┐
              │ Agent C │
              └─────────┘
```
