# Skill 技能系统

> agentscope-java Agent 能力层 | 动态技能加载

## 一句话

动态加载和管理 Agent 技能，支持运行时扩展能力。

---

## 核心概念

### Skill 定义

Skill 是一组相关工具和能力的集合，可以被 Agent 动态加载。

```
Skill {
  name: String              // 技能名称
  description: String       // 技能描述
  tools: List<Tool>        // 工具列表
  prompts: List<String>    // 提示模板
}
```

### Skill 分类

| 类型 | 说明 | 示例 |
|------|------|------|
| 文件操作 | 文件读写、搜索 | file_search, file_write |
| 代码执行 | 运行代码、脚本 | shell_command, python_exec |
| 网络访问 | HTTP 请求、搜索 | web_search, http_get |
| 数据处理 | 解析、转换 | json_parse, csv_read |

---

## Skill 加载

### 配置文件

```yaml
# skills/coding.yaml
name: coding
description: Code generation and execution
tools:
  - shell_command
  - file_read
  - file_write
prompts:
  - "You are an expert programmer..."
```

### 动态加载

```java
SkillLoader loader = new YamlSkillLoader();
Skill skill = loader.load("skills/coding.yaml");

// 注册到 Toolkit
toolkit.registerSkill(skill);
```

## 内置 Skill

|-------|------|
| `coding` | shell_command, file_read, file_write |
| `web` | web_search, http_get |
| `rag` | knowledge_search, document_read |

## 扩展机制

### 自定义 Skill

```java
public class MySkill implements Skill {
    @Override
    public String getName() {
        return "my_skill";
    }
    
    @Override
    public List<ToolSchema> getToolSchemas() {
        return List.of(...);
    }
    
    @Override
    public Object execute(String toolName, Map<String, Object> params) {
        // 执行工具
    }
}
```
