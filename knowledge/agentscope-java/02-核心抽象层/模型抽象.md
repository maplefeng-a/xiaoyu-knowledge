# Model 模型抽象

> agentscope-java 核心抽象层

## 一句话

统一的 LLM 接入层，支持 OpenAI、DashScope、Gemini、Ollama、Anthropic 等多种模型。

## 核心设计

- **统一接口**: ChatModel 抽象所有模型
- **配置驱动**: ExecutionConfig 控制采样参数
- **流式支持**: 基于 Reactor 的异步流式响应
- **工具调用**: 统一的 ToolSchema 机制

## 支持的模型

- OpenAI (GPT-4, GPT-3.5)
- DashScope (通义千问)
- Gemini (Google)
- Ollama (本地)
- Anthropic (Claude)
