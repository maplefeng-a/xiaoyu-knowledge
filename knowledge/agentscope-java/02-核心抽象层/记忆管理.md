# Memory 记忆管理

> agentscope-java 核心抽象层

## 一句话

会话内记忆（Memory）与跨会话长期记忆（LongTermMemory）的统一管理。

## 核心设计

- **接口分离**: Memory (会话内) + LongTermMemory (跨会话)
- **状态持久化**: StateModule 接口支持保存/加载
- **线程安全**: CopyOnWriteArrayList
- **异步优先**: 基于 Reactor Mono
