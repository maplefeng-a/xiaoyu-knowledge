# Session 会话管理

> agentscope-java 核心抽象层

## 一句话

Agent 状态持久化机制，支持跨重启的保存与恢复。

## 核心设计

- **简洁 API**: agent.saveTo() / agent.loadFrom()
- **多种存储**: JsonSession (文件) / InMemorySession (内存)
- **类型化状态**: StateModule 接口统一管理

## 存储格式

```
sessions/{sessionId}/
├── memory_messages.jsonl    # 消息列表
└── plan_notebook.json       # 计划状态
```
