# Tool 工具系统

> agentscope-java 核心抽象层

## 一句话

类型安全的工具调用机制，支持 POJO 方法自动转换为 Agent 可调用工具。

## 核心设计

- **POJO 即工具**: @Tool 注解标记方法
- **类型安全**: @ToolParam 定义参数schema
- **分组管理**: ToolGroup + ToolGroupManager
- **MCP 集成**: McpClientManager 支持外部工具

## 核心接口

```java
@Tool(name = "calculator")
public String calculator(
    @ToolParam(name = "expression") String expr
) { ... }
```
