# AgentScope 可观测性

> agentscope-java 的可观测性体系

## 概述

AgentScope 提供完整的可观测性支持，包括 Tracing、Studio 可视化等。

## 核心组件

### Tracing
- 基于 OpenTelemetry
- 分布式链路追踪
- Agent 执行全链路监控

### Studio
- AgentScope Studio 可视化
- 实时监控
- 调试支持
