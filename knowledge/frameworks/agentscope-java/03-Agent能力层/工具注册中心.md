# ToolRegistry 工具注册中心

> OpenClaw assistant-agent | 工具管理核心

## 一句话

集中式工具注册中心，管理本地工具和 MCP 工具的生命周期，支持默认工具 + Skill 专属工具。

---

## 核心设计

### 设计原则
- **共享工具包**: 每个会话共享一个工具包副本
- **分层管理**: 默认工具（始终可用） + Skill 工具（按需加载）
- **延迟绑定**: Skill 工具仅在需要时才注册到 Agent
- **快速查找**: HashMap 实现工具名称映射

### 依赖

```java
@Component
public class ToolRegistry {
    // AgentScope Core API
    private final List<AgentTool> tools;
    private final McpClientFactory mcpClientFactory;
    private final SkillProperties skillProperties;
    
    // Shared toolkit (default tools)
    private Toolkit sharedToolkit;
    
    // Tool name mapping
    private final Map<String, AgentTool> toolMap = new HashMap<>();
}
```

---

## 核心方法

### 1. 初始化（@PostConstruct）

```java
@PostConstruct
public void init() {
    sharedToolkit = new Toolkit();

    // Register default local tools
    SkillToolConfig defaultTools = skillProperties.getDefaultTools();
    for (String toolName : defaultTools.getLocal()) {
        AgentTool tool = toolMap.get(toolName);
        if (tool != null) {
            sharedToolkit.registerTool(tool);
            log.info("Registered default local tool: {}", toolName);
        }
    }

    // Register default MCP tools (ungrouped, always visible)
    for (SkillToolConfig.McpTool mcpConfig : defaultTools.getMcp()) {
        String serverName = mcpConfig.getServer();
        String toolName = mcpConfig.getName();

        McpClientWrapper client = mcpClientFactory.getOrCreateClient(serverName);
        if (client != null) {
            sharedToolkit.registration()
                .mcpClient(client)
                .enableTools(List.of(toolName))
                .apply();
            log.info("Registered default MCP tool '{}' from server '{}'", toolName, serverName);
        }
    }
}
```

### 2. 创建工具包（createToolkit）

```java
public Toolkit createToolkit() {
    return sharedToolkit.copy();
}
```

### 3. 工具查找（getTool)

```java
public AgentTool getTool(String toolName) {
    return toolMap.get(toolName);
}
```

## 工具分类

### 默认本地工具（7 个）
- **ReadTool** - 文件读取
- **WriteTool** - 文件写入
- **EditTool** - 文件编辑
- **BashTool** - Shell 命令执行
- **GlobTool** - 文件模式匹配
- **GrepTool** - 内容搜索

### 默认 MCP 工具（可配置）
- 通过 `McpClientFactory` 动态创建
- 支持多个 MCP 服务器
- 按需注册工具

- 共享给所有会话

### Skill 专属工具（延迟加载）
- 通过 `SkillLoader` 按 skill 配置加载
- 仅在需要时才注册到 Agent
- 不污染共享工具包

---

## 工具注册流程

```
1. Spring Boot 启动
   ↓
2. ToolRegistry.init()
   ├─ 注册默认本地工具
   ├─ 注册默认 MCP 工具
   ↓
3. Agent 创建时
   ├─ 调用 createToolkit()
   └─ 获取共享工具包副本
   ↓
4. Skill 加载时（如果需要）
   ├─ SkillLoader.createSkillBox()
   └─ 注册 Skill 专属工具到 Agent 的 toolkit
```

## 配置示例

```yaml
skill:
  default-tools:
    local:
      - read
      - write
      - edit
      - bash
      - glob
      - grep
    mcp:
      - server: filesystem
        name: read_file
      - server: filesystem
        name: write_file
```

## 最佳实践

### 1. 工具命名
- 使用简单、清晰的名称（read, write, edit）
- 避免使用类名作为工具名
- 保持一致性

### 2. MCP 服务器管理
- 使用 `McpClientFactory` 统一管理
- 支持懒加载（按需创建）
- 缓存客户端连接

### 3. 性能优化
- 共享工具包减少内存占用
- HashMap 加速工具查找
- 避免重复注册
