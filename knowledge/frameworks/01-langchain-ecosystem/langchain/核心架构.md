## 核心概念

### 1. Runnable（可运行接口）

```python
from langchain_core.runnables import Runnable

class MyRunnable(Runnable):
    def invoke(self, input: dict) -> str:
        return "Hello, " + input["name"]
    
    def batch(self, inputs: list[dict]) -> list[str]:
        return [self.invoke(inp) for inp in inputs]
    
    def stream(self, input: dict):
        yield "Hello, "
        yield input["name"]
```

**特性：**
- **invoke** - 同步调用
- **batch** - 批量调用
- **stream** - 流式调用
- **ainvoke** - 异步调用

### 2. Callbacks（回调）

```python
from langchain_core.callbacks import BaseCallbackHandler

class MyCallbackHandler(BaseCallbackHandler):
    def on_llm_start(self, serialized, prompts, **kwargs):
        print("LLM started")
    
    def on_llm_end(self, response, **kwargs):
        print("LLM finished")
    
    def on_tool_start(self, serialized, input_str, **kwargs):
        print("Tool started")
    
    def on_tool_end(self, output, **kwargs):
        print("Tool finished")
```

**事件类型：**
- **on_llm_start/end** - LLM 调用
- **on_chain_start/end** - 链调用
- **on_tool_start/end** - 工具调用
- **on_agent_action/finish** - 智能体动作

## 使用示例

### 1. 简单 LLM 链

```python
from langchain_openai import ChatOpenAI
from langchain_core.prompts import ChatPromptTemplate
from langchain_core.output_parsers import StrOutputParser

# 创建模型
llm = ChatOpenAI(model="gpt-4")

# 创建提示词
prompt = ChatPromptTemplate.from_template("Tell me a joke about {topic}")

# 创建输出解析器
output_parser = StrOutputParser()

# 构建链
chain = prompt | llm | output_parser

# 调用
result = chain.invoke({"topic": "ice cream"})
print(result)
```

### 2. 带记忆的对话

```python
from langchain.memory import ConversationBufferMemory
from langchain.chains import ConversationChain

# 创建记忆
memory = ConversationBufferMemory()

# 创建对话链
conversation = ConversationChain(
    llm=llm,
    memory=memory,
    verbose=True
)

# 多轮对话
conversation.predict(input="Hi, I'm Alice")
conversation.predict(input="What's my name?")
```

### 3. Agent with Tools

```python
from langchain.agents import create_openai_functions_agent, AgentExecutor
from langchain.tools import Tool

# 定义工具
def get_word_length(word: str) -> int:
    return len(word)

tools = [
    Tool(
        name="get_word_length",
        func=get_word_length,
        description="Returns the length of a word"
    )
]

# 创建 Agent
agent = create_openai_functions_agent(llm, tools)
agent_executor = AgentExecutor(agent=agent, tools=tools)

# 调用
result = agent_executor.invoke({"input": "How long is the word 'hello'?"})
```

## 最佳实践

### 1. 使用 Runnable 接口
- 优先使用 `|` 操作符构建链
- 利用 `invoke`/`batch`/`stream` 方法
- 使用 `ainvoke` 进行异步调用

### 2. 记忆管理
- 对话用 `ConversationBufferMemory`
- 长对话用 `ConversationSummaryMemory`
- 知识图谱用 `ConversationKGMemory`

### 3. 工具定义
- 继承 `BaseTool` 类
- 提供 `name` 和 `description`
- 实现 `_run` 方法

### 4. 回调使用
- 继承 `BaseCallbackHandler`
- 实现需要的回调方法
- 通过 `callbacks` 参数传递

## 技术亮点

### 1. Runnable LCEL（LangChain Expression Language）
- **链式调用** - `prompt | llm | output_parser`
- **统一接口** - `invoke`/`batch`/`stream`
- **可组合** - 支持复杂的 DAG 结构

### 2. 回调系统
- **生命周期钩子** - start/end/error
- **细粒度追踪** - LLM/Chain/Tool/Agent
- **可扩展** - 自定义回调处理器

### 3. 提示词工程
- **模板化** - 支持变量插值
- **少样本** - FewShotPromptTemplate
- **示例选择** - SemanticSimilarityExampleSelector
