
# 扩展机制

> agentscope-java 扩展能力

## 一句话

三大核心扩展点：Tool（8+）、Agent（8+）、Model（7+），支持注解、继承、接口等多种扩展方式。

## 核心扩展点

### Tool 扩展
| 扩展点 | 方式 | 用途 |
|--------|------|------|
| `@Tool` 注解 | 声明式 | POJO 方法转工具 |
| `AgentTool` 接口 | 编程式 | 自定义工具行为 |
| `ToolGroup` | 功能扩展 | 动态工具组管理 |
| `McpClientManager` | 协议扩展 | MCP 工具集成 |

### Agent 扩展
| 扩展点 | 方式 | 用途 |
|--------|------|------|
| `AgentBase` | 继承 | 自定义 Agent 逻辑 |
| `Hook` 接口 | 事件订阅 | 监听/拦截执行 |
| `StateModule` | 机制扩展 | 状态持久化 |
| `CallableAgent` | 混合扩展 | Agent as Tool |

### Model 扩展
| 扩展点 | 方式 | 用途 |
|--------|------|------|
| `ChatModelBase` | 继承 | 集成新 LLM |
| `ChatMessageFormatter` | 策略扩展 | 消息格式适配 |
| `ToolChoice` | 策略扩展 | 工具调用控制 |

## 扩展示例

```java
// Tool 扩展
@Tool(name = "custom_tool")
public String customMethod(@ToolParam String param) { ... }

// Agent 扩展
class CustomAgent extends AgentBase {
    protected Mono<Msg> executeCoreLogic(Msg msg) { ... }
}

// Model 扩展
class CustomModel extends ChatModelBase {
    protected Flux<ChatResponse> doStream(...) { ... }
}
```

## 常见扩展模块

- `extensions-rag-*` - RAG 能力
- `extensions-reme` - ReMe 长期记忆
- `extensions-session-*` - 会话存储
- `spring-boot-starters` - Spring Boot 集成
