
# 框架集成总览

> agentscope-java Java 框架集成扩展

## 一句话

提供三种主流 Java 框架集成：Spring Boot、Quarkus、Micronaut，通过 Starter/Extension 简化 AgentScope 集成。

## 三种框架集成对比

| 特性 | Spring Boot | Quarkus | Micronaut |
|------|-------------|---------|-----------|
| **启动速度** | 中等 | 极快（Native） | 快（Native） |
| **内存占用** | 中等 | 极低 | 低 |
| **生态成熟度** | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ |
| **响应式支持** | WebFlux | Vert.x | HTTP Client |
| **云原生** | 一般 | 优秀 | 优秀 |
| **适用场景** | 企业应用 | 云原生/Serverless | 微服务 |

## Spring Boot Starter

### Maven 依赖

```xml
<dependency>
    <groupId>io.agentscope</groupId>
    <artifactId>agentscope-spring-boot-starter</artifactId>
    <version>${agentscope.version}</version>
</dependency>
```

### 配置示例

```yaml
# application.yml
agentscope:
  model:
    type: dashscope
    api-key: ${DASHSCOPE_API_KEY}
    model-name: qwen-plus
  agent:
    name: Assistant
    sys-prompt: You are a helpful assistant.
```

### 使用案例

```java
@SpringBootApplication
public class Application {
    public static void main(String[] args) {
        SpringApplication.run(Application.class, args);
    }
}

@RestController
class AgentController {
    
    @Autowired
    private ReActAgent agent;
    
    @PostMapping("/chat")
    public Mono<String> chat(@RequestBody String message) {
        return agent.call(Msg.userText(message))
            .map(Msg::getTextContent);
    }
}
```

## Quarkus Extension

### Maven 依赖

```xml
<dependency>
    <groupId>io.agentscope</groupId>
    <artifactId>agentscope-quarkus-extension</artifactId>
    <version>${agentscope.version}</version>
</dependency>
```

### 配置示例

```yaml
# application.properties
agentscope.model.type=dashscope
agentscope.model.api-key=${DASHSCOPE_API_KEY}
agentscope.model.model-name=qwen-plus
```

### 使用案例

```java
@QuarkusMain
public class Application {
    public static void main(String[] args) {
        Quarkus.run(args);
    }
}

@Path("/chat")
class AgentResource {
    
    @Inject
    ReActAgent agent;
    
    @POST
    @Produces(MediaType.TEXT_PLAIN)
    public Uni<String> chat(String message) {
        return Uni.createFrom().publisher(
            agent.call(Msg.userText(message))
                .map(Msg::getTextContent)
        );
    }
}
```

## Micronaut Extension

### Maven 依赖

```xml
<dependency>
    <groupId>io.agentscope</groupId>
    <artifactId>agentscope-micronaut-extension</artifactId>
    <version>${agentscope.version}</version>
</dependency>
```

### 配置示例

```yaml
# application.yml
agentscope:
  model:
    type: dashscope
    api-key: ${DASHSCOPE_API_KEY}
    model-name: qwen-plus
```

### 使用案例

```java
@MicronautTest
class AgentTest {
    
    @Inject
    ReActAgent agent;
    
    @Test
    void testChat() {
        String response = agent.call(Msg.userText("Hello"))
            .map(Msg::getTextContent)
            .block();
        assertNotNull(response);
    }
}
```

## 选择建议

| 场景 | 推荐框架 |
|------|---------|
| **企业应用** | Spring Boot（生态丰富） |
| **Serverless** | Quarkus（启动快、内存低） |
| **微服务** | Micronaut 或 Quarkus |
| **已有 Spring 项目** | Spring Boot Starter |
| **响应式应用** | Spring WebFlux + AgentScope |
