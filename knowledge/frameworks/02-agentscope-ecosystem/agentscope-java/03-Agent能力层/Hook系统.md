# Hook 钩子系统

> agentscope-java Agent 能力层 | 统一事件模型

## 一句话

统一事件模型，在 Agent 执行的特定阶段监控和修改行为，实现可观测性和扩展性。

---

## Hook 接口

```java
public interface Hook {
    // 处理事件（统一入口）
    <T extends HookEvent> Mono<T> onEvent(T event);
    
    // 优先级（值越小越先执行）
    default int priority() {
        return 100;
    }
}
```

---

## 优先级范围

| 范围 | 用途 | 示例 |
|------|------|------|
| 0-50 | 关键系统 Hook | 认证、安全 |
| 51-100 | 高优先级 | 验证、预处理 |
| 101-500 | 普通优先级 | 业务逻辑 |
| 501-1000 | 低优先级 | 日志、指标 |

---

## 事件类型

### 生命周期事件

| 事件 | 时机 | 可修改 | 用途 |
|------|------|--------|------|
| `PreCallEvent` | Agent 调用前 | ❌ | 记录开始 |
| `PostCallEvent` | Agent 调用后 | ✅ | 修改最终响应 |

### 推理事件

| 事件 | 时机 | 可修改 | 用途 |
|------|------|--------|------|
| `PreReasoningEvent` | LLM 推理前 | ✅ | 注入提示 |
| `PostReasoningEvent` | LLM 推理后 | ✅ | 修改结果 |
| `ReasoningChunkEvent` | 推理流式输出 | ❌ | 实时显示 |

### 工具事件

| 事件 | 时机 | 可修改 | 用途 |
|------|------|--------|------|
| `PreActingEvent` | 工具执行前 | ✅ | 参数校验、注入 |
| `PostActingEvent` | 工具执行后 | ✅ | 修改结果 |
| `ActingChunkEvent` | 工具流式输出 | ❌ | 实时显示 |

### 其他事件

| 事件 | 时机 | 可修改 | 用途 |
|------|------|--------|------|
| `PreSummaryEvent` | 生成总结前 | ✅ | 自定义总结 |
| `PostSummaryEvent` | 生成总结后 | ✅ | 修改总结 |
| `ErrorEvent` | 发生错误 | ❌ | 错误处理、重试 |

## 可修改事件

### 修改输入消息

```java
case PreReasoningEvent e -> {
    List<Msg> msgs = new ArrayList<>(e.getInputMessages());
    msgs.add(0, Msg.builder()
        .role(MsgRole.SYSTEM)
        .textContent("Think step by step")
        .build());
    e.setInputMessages(msgs);
    yield Mono.just(e);
}
```

### 修改工具参数

```java
case PreActingEvent e -> {
    ToolUseBlock toolUse = e.getToolUse();
    Map<String, Object> input = new HashMap<>(toolUse.getInput());
    input.put("authToken", "secret");  // 注入认证
    e.setToolUse(ToolUseBlock.builder()
        .id(toolUse.getId())
        .name(toolUse.getName())
        .input(input)
        .build());
    yield Mono.just(e);
}
```

### 修改工具结果

```java
case PostActingEvent e -> {
    ToolResultBlock result = e.getToolResult();
    // 过滤敏感信息
    String filtered = result.getOutput().getTextContent()
        .replaceAll("password=\\S+", "password=***");
    e.setToolResult(ToolResultBlock.builder()
        .id(result.getId())
        .name(result.getName())
        .output(TextBlock.builder().text(filtered).build())
        .build());
    yield Mono.just(e);
}
```

### 修改最终响应

```java
case PostCallEvent e -> {
    Msg response = e.getResponse();
    Msg modified = Msg.builder()
        .role(response.getRole())
        .textContent(response.getTextContent() + "\n\n---\nGenerated by AgentScope")
        .build();
    e.setResponse(modified);
    yield Mono.just(e);
}
```

## 使用示例

### 日志 Hook

```java
Hook loggingHook = new Hook() {
    @Override
    public <T extends HookEvent> Mono<T> onEvent(T event) {
        return switch (event) {
            case PreCallEvent e -> {
                log.info("Agent {} started", e.getAgent().getName());
                yield Mono.just(event);
            }
            case PostCallEvent e -> {
                log.info("Agent completed: {}", e.getResponse().getTextContent());
                yield Mono.just(event);
            }
            case ErrorEvent e -> {
                log.error("Agent error", e.getError());
                yield Mono.just(event);
            }
            default -> Mono.just(event);
        };
    }
};

agent.addHook(loggingHook);
```

### 流式显示 Hook

```java
Hook streamingHook = new Hook() {
    @Override
    public <T extends HookEvent> Mono<T> onEvent(T event) {
        return switch (event) {
            case ReasoningChunkEvent e -> {
                System.out.print(e.getIncrementalChunk());
                yield Mono.just(event);
            }
            case ActingChunkEvent e -> {
                System.out.println("🔧 " + e.getToolName() + ": " + e.getIncrementalChunk());
                yield Mono.just(event);
            }
            default -> Mono.just(event);
        };
    }
};
```

### 高优先级 Hook

```java
Hook authHook = new Hook() {
    @Override
    public int priority() {
        return 10;  // 最先执行
    }
    
    @Override
    public <T extends HookEvent> Mono<T> onEvent(T event) {
        return switch (event) {
            case PreActingEvent e -> {
                // 在所有其他 Hook 之前注入认证
                injectAuth(e.getToolUse());
                yield Mono.just(event);
            }
            default -> Mono.just(event);
        };
    }
};
```
