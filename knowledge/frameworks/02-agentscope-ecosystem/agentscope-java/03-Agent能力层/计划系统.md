# Plan 计划系统

> agentscope-java Agent 能力层 | 任务分解与计划

## 一句话

任务分解和计划管理系统，支持复杂任务的子任务拆分和状态跟踪。

---

## 核心概念

### PlanNotebook

```java
PlanNotebook {
  goal: String                    // 总目标
  subTasks: List<SubTask>        // 子任务列表
  currentState: PlanState        // 当前状态
}
```

### SubTask

```java
SubTask {
  id: String                     // 子任务 ID
  description: String            // 任务描述
  state: SubTaskState            // 状态
  dependencies: List<String>     // 依赖的任务 ID
  result: String                 // 执行结果
}
```

### SubTaskState

| 状态 | 说明 |
|------|------|
| `PENDING` | 待执行 |
| `IN_PROGRESS` | 执行中 |
| `COMPLETED` | 已完成 |
| `FAILED` | 失败 |
| `SKIPPED` | 跳过 |

---

## 使用示例

### 创建计划

```java
PlanNotebook plan = PlanNotebook.builder()
    .goal("Build a REST API")
    .subTask("Design API schema")
    .subTask("Implement endpoints")
    .subTask("Write tests")
    .subTask("Deploy")
    .build();
```

### 执行计划

```java
for (SubTask task : plan.getSubTasks()) {
    if (task.getState() == SubTaskState.PENDING) {
        task.setState(SubTaskState.IN_PROGRESS);
        String result = agent.call(task.getDescription()).block().getTextContent();
        task.setResult(result);
        task.setState(SubTaskState.COMPLETED);
    }
}
```

---

## 持久化

```java
// 保存到 Session
session.save(sessionKey, "plan_notebook", plan);

// 恢复
Optional<PlanNotebook> loaded = session.get(sessionKey, "plan_notebook", PlanNotebook.class);
```
