
# 中断机制

> 03-Agent能力层 | 任务中断处理

## 一句话

提供中断上下文信息，记录中断来源、时间、用户消息和待处理工具调用。
## 核心组件

| 组件 | 作用 |
|------|------|
| InterruptSource | 中断来源枚举：USER/TOOL/SYSTEM |
| InterruptContext | 中断上下文，包含 source/timestamp/userMessage/pendingToolCalls |
## InterruptSource

| 来源 | 说明 |
|------|------|
| USER | 用户主动中断 |
| TOOL | 工具执行逻辑中断 |
| SYSTEM | 系统中断（超时/资源限制等） |
## InterruptContext

```java
InterruptContext context = InterruptContext.builder()
    .source(InterruptSource.USER)
    .userMessage(userMsg)
    .pendingToolCalls(toolCalls)
    .build();
```

## 使用场景

- 用户取消正在执行的任务
- 工具执行超时
- 系统资源限制
