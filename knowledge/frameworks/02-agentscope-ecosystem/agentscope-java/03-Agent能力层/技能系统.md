# 技能系统

> 03-Agent能力层 | 可动态加载的能力单元

## 一句话

技能（Skill）是包含指令、资源和工具的能力单元，可被 Agent 动态加载和激活。

## 核心组件

| 组件 | 作用 |
|------|------|
| AgentSkill | 技能定义，包含 name/description/content/resources |
| SkillRegistry | 技能注册表，存储技能和激活状态 |
| SkillBox | 技能箱，管理注册/激活/工具绑定/代码执行 |
| RegisteredSkill | 已注册技能包装器，跟踪激活状态 |

## AgentSkill 结构

```java
public class AgentSkill {
    private final String name;           // 技能名称
    private final String description;    // 技能描述
    private final String skillContent;   // 技能指令（Markdown）
    private final Map<String, String> resources;  // 资源文件
    private final String source;         // 来源标识
}
```

## SkillBox 核心方法

```java
// 注册技能
skillBox.registerSkill(skill);

// 流式注册（技能 + 工具）
skillBox.registration()
    .skill(skill)
    .tool(toolObject)
    .apply();

// 激活/停用技能
skillBox.deactivateAllSkills();
skillBox.isSkillActive(skillId);

// 代码执行配置
skillBox.codeExecution()
    .workDir("/path/to/workdir")
    .withShell()
    .withRead()
    .withWrite()
    .enable();
```

## Skill vs Tool

| 维度 | Skill | Tool |
|------|-------|------|
| 定义 | 技能 = 指令 + 资源 + 工具 | 单一功能函数 |
| 粒度 | 粗粒度，可包含多个工具 | 细粒度，单一功能 |
| 激活 | 需要显式加载 | 默认可用 |
| 内容 | Markdown + 资源文件 | Java 方法 |

## 使用示例

```java
// 创建技能
AgentSkill skill = AgentSkill.builder()
    .name("code_review")
    .description("Code review skill")
    .skillContent("# Code Review\nReview code...")
    .addResource("checklist.md", "## Checklist...")
    .build();

// 注册技能
SkillBox skillBox = new SkillBox(toolkit);
skillBox.registerSkill(skill);

// 注册技能 + 工具
skillBox.registration()
    .skill(skill)
    .tool(new ReviewTool())
    .apply();

// 配置代码执行
skillBox.codeExecution()
    .workDir("/workspace")
    .withShell()
    .withRead()
    .withWrite()
    .enable();
```
