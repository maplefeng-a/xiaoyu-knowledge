
# Tool 工具系统

> agentscope-java 核心抽象层 | 类型安全的工具调用机制

## 一句话

POJO 方法通过注解自动转换为 Agent 可调用工具，支持分组管理、MCP 集成、异步执行。

---

## 核心注解

### @Tool - 方法级注解

```java
@Target(ElementType.METHOD)
@Retention(RetentionPolicy.RUNTIME)
public @interface Tool {
    String name() default "";          // 工具名（默认方法名）
    String description() default "";   // 描述（发送给 LLM）
    Class<?> resultConverter() default void.class;  // 结果转换器
}
```

### @ToolParam - 参数级注解

```java
@Target(ElementType.PARAMETER)
@Retention(RetentionPolicy.RUNTIME)
public @interface ToolParam {
    String name();                     // 参数名（必需）
    String description() default "";   // 描述
    boolean required() default true;   // 是否必需
}
```

## Toolkit 架构

### 组件职责

```
Toolkit (Facade)
├── ToolRegistry           // 工具注册和查找
├── ToolGroupManager       // 分组 CRUD 和激活管理
├── ToolSchemaProvider     // 带分组过滤的 Schema 生成
├── McpClientManager       // MCP 客户端生命周期
├── MetaToolFactory        // 元工具（如 reset_equipped_tools）
├── ToolSchemaGenerator    // JSON Schema 生成
├── ToolMethodInvoker      // 反射调用和参数转换
├── ToolResultConverter    // 结果 → ToolResultBlock
└── ToolExecutor           // 并行/串行执行
```

## 工具定义

### 同步工具

```java
public class WeatherTools {
    @Tool(description = "Get current weather for a city")
    public String getWeather(
        @ToolParam(name = "city", description = "City name, e.g., 'Beijing'") 
        String city
    ) {
        return "Sunny, 25°C";
    }
}
```

### 异步工具

```java
public class ApiTools {
    @Tool(description = "Fetch data from API endpoint")
    public Mono<String> fetchData(
        @ToolParam(name = "url", description = "API URL (must start with https://api.myservice.com)")
        String url
    ) {
        // 安全校验
        if (!url.startsWith("https://api.myservice.com")) {
            return Mono.just("Error: URL not allowed");
        }
        return webClient.get()
            .uri(url)
            .retrieve()
            .bodyToMono(String.class)
            .timeout(Duration.ofSeconds(10));
    }
}
```

### 流式工具（ToolEmitter）

```java
@Tool(description = "Long running task with progress")
public void longTask(
    @ToolParam(name = "task") String task,
    ToolEmitter emitter  // 框架自动注入，无需 @ToolParam
) {
    emitter.emit("Starting...");
    // 处理...
    emitter.emit("Progress: 50%");
    // 完成
    emitter.emit("Done");
}
```

## 工具注册

### 方式 1: 直接注册

```java
Toolkit toolkit = new Toolkit();
toolkit.registerTool(new WeatherTools());
toolkit.registerTool(new ApiTools());
```

### 方式 2: Fluent API

```java
toolkit.registration()
    .tool(new WeatherTools())
    .group("weather")
    .presetParameters(Map.of("apiKey", "secret"))  // 预设参数
    .apply();
```

### 方式 3: MCP 客户端

```java
McpClientWrapper mcpClient = McpClientBuilder.stdio()
    .command("npx")
    .args("-y", "@modelcontextprotocol/server-filesystem", "/path")
    .build();

toolkit.registration()
    .mcpClient(mcpClient)
    .enableTools(List.of("read_file", "write_file"))
    .group("filesystem")
    .apply();
```

## 工具分组

### 创建和管理

```java
// 创建分组
toolkit.createToolGroup("database", "Database operations");

// 注册到分组
toolkit.registerTool(new DbTools(), "database");

// 激活分组（只有激活的工具对 Agent 可见）
toolkit.setActiveGroups(Set.of("database", "weather"));

// 获取激活的工具 Schema
List<ToolSchema> schemas = toolkit.getActiveToolSchemas();

// 动态切换工具集
toolkit.setActiveGroups(Set.of("filesystem"));  // 只保留文件工具
```

### 元工具

```java
// 内置元工具：reset_equipped_tools
// 允许 Agent 动态切换工具组
toolkit.setActiveGroups(Set.of("meta"));  // 启用元工具
```

## 工具执行

### 单个工具

```java
ToolUseBlock toolUse = response.getFirstContentBlock(ToolUseBlock.class);
Mono<ToolResultBlock> result = toolkit.executeTool(toolUse);

result.subscribe(toolResult -> {
    String output = toolResult.getOutput();
});
```

### 批量执行（并行）

```java
List<ToolUseBlock> toolCalls = response.getContentBlocks(ToolUseBlock.class);
Mono<List<ToolResultBlock>> results = toolkit.executeTools(toolCalls);

results.subscribe(toolResults -> {
    // 所有工具执行完成
});
```

### 挂起机制

```java
@Tool(description = "Execute external command")
public String executeCommand(
    @ToolParam(name = "command") String command
) throws ToolSuspendException {
    // 需要用户确认，挂起执行
    throw new ToolSuspendException("Please confirm command execution");
}
```

## 特殊机制

### ToolEmitter

```java
// 框架自动注入，用于流式输出
public void streamingTool(ToolEmitter emitter) {
    emitter.emit("Step 1");
    emitter.emit("Step 2");
    emitter.emit("Done");
}
```

### PresetParameters

```java
// 预设参数，隐藏敏感信息
toolkit.registration()
    .tool(new ApiTools())
    .presetParameters(Map.of(
        "fetchData", Map.of("apiKey", "secret-key")  // 自动注入
    ))
    .apply();
```

### ToolSuspendException

```java
// 挂起工具，等待外部处理
// GenerateReason.TOOL_SUSPENDED
throw new ToolSuspendException("Waiting for user confirmation");
```

## 与 Model 集成

```java
// 1. 获取工具 Schema
List<ToolSchema> toolSchemas = toolkit.getActiveToolSchemas();

// 2. 传给 Model
Flux<ChatResponse> responses = model.stream(messages, toolSchemas, null);

// 3. 提取工具调用
responses
    .filter(resp -> resp.hasContentBlocks(ToolUseBlock.class))
    .flatMap(resp -> {
        List<ToolUseBlock> toolCalls = resp.getContentBlocks(ToolUseBlock.class);
        return toolkit.executeTools(toolCalls);
    })
    .subscribe(result -> {
        // 处理工具结果
    });
```
