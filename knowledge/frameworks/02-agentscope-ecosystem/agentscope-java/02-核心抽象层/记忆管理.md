
# Memory 记忆管理

> agentscope-java 核心抽象层 | 会话内 + 跨会话记忆

## 一句话

会话内记忆与跨会话长期记忆的统一管理，支持自动/手动两种控制模式。

---

## 两种记忆类型

| 类型 | 作用域 | 接口 | 用途 |
|------|--------|------|------|
| **Memory** | 会话内 | 同步 API | 当前对话历史 |
| **LongTermMemory** | 跨会话 | 异步 API (Mono) | 持久化记忆 |

---

## Memory 接口（会话内）

```java
public interface Memory extends StateModule {
    void addMessage(Msg message);       // 添加消息
    List<Msg> getMessages();            // 获取所有消息
    void deleteMessage(int index);      // 删除指定位置消息
    void clear();                       // 清空记忆
}
```

### 实现：InMemoryMemory

```java
// 默认实现，内存存储
Memory memory = new InMemoryMemory();

// 使用
memory.addMessage(userMsg);
memory.addMessage(agentMsg);
List<Msg> history = memory.getMessages();
```

### StateModule 支持

Memory 继承 StateModule，支持状态持久化：

```java
// 保存状态
Map<String, Object> state = memory.getState();

// 恢复状态
memory.setState(state);
```

## LongTermMemory 接口（跨会话）

```java
public interface LongTermMemory {
    // 记录消息到长期记忆（异步）
    Mono<Void> record(List<Msg> msgs);
    
    // 检索相关记忆（异步）
    Mono<String> retrieve(Msg msg);
}
```

### 实现类

| 实现 | 说明 | 扩展模块 |
|------|------|----------|
| `Mem0LongTermMemory` | Mem0 服务集成 | `agentscope-extensions-mem0` |
| `ReMeLongTermMemory` | ReMe 服务集成 | `agentscope-extensions-reme` |
| `AutoContextMemory` | 自动上下文记忆 | `agentscope-extensions-autocontext-memory` |

---

## LongTermMemoryMode

### 模式说明

| 模式 | 自动检索 | 自动记录 | Agent 工具 | 适用场景 |
|------|----------|----------|------------|----------|
| `STATIC_CONTROL` | ✓ | ✓ | ✗ | 简单 Agent，全自动 |
| `AGENT_CONTROL` | ✗ | ✗ | ✓ | 高级 Agent，完全自主 |
| `BOTH` | ✓ | ✓ | ✓ | **推荐默认** |

### 工作流程

#### STATIC_CONTROL 模式

```
用户消息 → retrieve() → 注入 System Prompt
                ↓
            Agent 推理
                ↓
            Agent 回复 → record() → 持久化
```

#### AGENT_CONTROL 模式

```
用户消息 → Agent 决定是否调用记忆工具
                ↓
        save_memory() / search_memory()
                ↓
            Agent 回复
```

#### BOTH 模式

```
用户消息 → retrieve() 自动检索
                ↓
        Agent 可调用记忆工具
                ↓
            Agent 回复 → record() 自动记录
```

## 使用示例

### 基础配置

```java
// 创建长期记忆
LongTermMemory longTermMemory = Mem0LongTermMemory.builder()
    .apiKey(System.getenv("MEM0_API_KEY"))
    .userId("user_123")
    .build();

// 配置 Agent
ReActAgent agent = ReActAgent.builder()
    .name("Assistant")
    .model(model)
    .memory(new InMemoryMemory())           // 会话内
    .longTermMemory(longTermMemory)          // 跨会话
    .longTermMemoryMode(LongTermMemoryMode.BOTH)
    .build();
```

### AGENT_CONTROL 模式

```java
// Agent 通过工具控制记忆
ReActAgent agent = ReActAgent.builder()
    .name("Assistant")
    .model(model)
    .longTermMemory(memory)
    .longTermMemoryMode(LongTermMemoryMode.AGENT_CONTROL)
    .build();

// Agent 可调用工具：
// - save_memory(content) - 保存记忆
// - search_memory(query) - 搜索记忆
```
