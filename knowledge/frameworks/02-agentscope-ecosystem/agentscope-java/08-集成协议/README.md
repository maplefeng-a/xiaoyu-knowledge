# AgentScope 集成协议

> agentscope-java 支持的集成协议

## 概述

AgentScope 支持多种协议，实现与外部系统的集成。

## 支持的协议

### MCP (Model Context Protocol)
- 工具调用协议
- 连接 MCP 生态

### A2A (Agent-to-Agent)
- 多智能体协作协议
- 服务发现与注册

### AGUI
- Agent-UI 交互协议
- 前端集成支持
