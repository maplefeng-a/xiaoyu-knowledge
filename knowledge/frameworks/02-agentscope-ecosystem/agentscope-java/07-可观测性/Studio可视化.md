# Studio 可视化

> AgentScope Studio 可视化监控平台

## 一句话

AgentScope Studio 是一个可视化监控平台，提供 Agent 执行流程可视化、实时监控、调试支持等功能。

## 核心功能

| 功能 | 说明 |
|------|------|
| **执行流程图** | 可视化 Reasoning → Acting → Summary 流程 |
| **消息查看** | 查看每条消息的详细内容 |
| **工具调用** | 查看工具调用参数和结果 |
| **Hook 事件** | 查看 Hook 触发情况 |
| **性能统计** | 执行时间、Token 消耗统计 |
| **断点调试** | 支持在指定位置暂停 |

## 使用案例

### 启动 Studio

```java
// 方式一：独立启动
StudioServer studio = StudioServer.builder()
    .port(8081)
    .build();
studio.start();

// 方式二：Spring Boot 自动配置
// 添加依赖后自动启动
```

### 访问界面

```
http://localhost:8081
```

## 优势与劣势

### 优势 ✅

| 优势 | 说明 |
|------|------|
| **可视化** | 直观展示执行流程 |
| **实时监控** | 实时查看 Agent 状态 |
| **调试友好** | 支持断点、日志查看 |
| **性能分析** | 统计执行时间和 Token |

### 劣势 ⚠️

| 劣势 | 说明 |
|------|------|
| **资源占用** | 需要额外资源运行 |
| **生产限制** | 主要用于开发调试 |
