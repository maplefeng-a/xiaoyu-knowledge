## 与 AgentScope-Java 对比

### 相似之处
- 都有 Chain/Pipeline 概念
- 都有 Memory 系统
- 都有 Tool 系统

### 差异之处
- **LangChain**: 通用 LLM 应用框架，强调链式调用
- **AgentScope-Java**: 智能体框架，强调 Agent 能力
