
# 基础设施扩展总览

> agentscope-java 基础设施扩展集合

## 一句话

提供四种基础设施扩展：Session-Redis、Session-MySQL、Scheduler、Studio，支持会话持久化、任务调度、可视化监控。

## 四种基础设施扩展

| 扩展 | 功能 | 适用场景 |
|------|------|---------|
| **Session-Redis** | Redis 会话存储 | 分布式部署、高可用 |
| **Session-MySQL** | MySQL 会话存储 | 持久化、关系查询 |
| **Scheduler** | 任务调度 | 定时任务、延迟执行 |
| **Studio** | 可视化监控 | 开发调试、运维监控 |

---

## Session-Redis

### 核心特点

- **高性能**：Redis 内存存储
- **分布式**：支持多实例共享会话
- **过期策略**：支持 TTL

### 使用案例

```java
// 创建 Redis Session 存储
RedisSessionStore sessionStore = RedisSessionStore.builder()
    .host("localhost")
    .port(6379)
    .password("password")
    .keyPrefix("agentscope:session:")
    .ttl(Duration.ofHours(24))
    .build();

// 与 Agent 集成
ReActAgent agent = ReActAgent.builder()
    .name("Assistant")
    .model(model)
    .sessionStore(sessionStore)
    .build();

// 保存状态
agent.saveTo(session, sessionKey);

// 恢复状态
agent.loadFrom(session, sessionKey);
```

## Session-MySQL

### 核心特点

- **持久化**：数据可靠存储
- **关系查询**：支持复杂查询
- **事务支持**：ACID 事务

### 使用案例

```java
// 创建 MySQL Session 存储
MysqlSessionStore sessionStore = MysqlSessionStore.builder()
    .jdbcUrl("jdbc:mysql://localhost:3306/agentscope")
    .username("root")
    .password("password")
    .tableName("agent_sessions")
    .build();

// 使用方式与 Redis 相同
```

## Scheduler

### 核心特点

- **定时任务**：Cron 表达式
- **延迟执行**：指定延迟后执行
- **持久化**：任务状态持久化

### 使用案例

```java
// 创建调度器
AgentScheduler scheduler = AgentScheduler.builder()
    .agent(myAgent)
    .jobStore(new InMemoryJobStore())  // 或 JdbcJobStore
    .build();

// 定时任务（每天 9:00 执行）
scheduler.scheduleJob(
    "daily-report",
    "0 0 9 * * ?",
    () -> agent.call(Msg.userText("生成日报")).subscribe()
);

// 延迟执行（30 秒后）
scheduler.scheduleOnce(
    "delayed-task",
    Duration.ofSeconds(30),
    () -> agent.call(Msg.userText("延迟任务")).subscribe()
);

// 启动调度器
scheduler.start();
```

## Studio

### 核心特点

- **可视化**：Agent 执行流程可视化
- **实时监控**：实时查看 Agent 状态
- **调试支持**：断点、日志查看

### 使用案例

```java
// 启动 Studio（通常在 Spring Boot 中自动配置）
StudioServer studio = StudioServer.builder()
    .port(8081)
    .agent(myAgent)
    .build();

studio.start();

// 访问 http://localhost:8081 查看 Agent 执行流程
```

### Studio 功能

- **执行流程图**：可视化 Reasoning → Acting → Summary
- **消息查看**：查看每条消息的内容
- **工具调用**：查看工具调用和结果
- **Hook 事件**：查看 Hook 触发情况
- **性能统计**：执行时间、Token 消耗

## 选择建议

| 场景 | 推荐扩展 |
|------|---------|
| **单机开发** | InMemorySession（默认） |
| **分布式部署** | Session-Redis |
| **持久化需求** | Session-MySQL |
| **定时任务** | Scheduler |
| **开发调试** | Studio |
| **生产监控** | Studio + Prometheus |
