## 核心概念

### 1. Agent 抽象

```python
from agentscope.agent import AgentBase

class MyAgent(AgentBase):
    def __init__(self, name: str, **kwargs):
        super().__init__(name, **kwargs)
        self.memory = TemporaryMemory()
    
    def reply(self, x: Msg) -> Msg:
        # 处理消息
        response = self.model(x)
        self.memory.add(x, response)
        return response
```

### 2. Message 抽象

```python
from agentscope.message import Msg

# 创建消息
msg = Msg(
    name="user",
    content="Hello, how are you?",
    role="user"
)

# 多模态消息
msg = Msg(
    name="user",
    content=[
        TextBlock(text="What's in this image?"),
        ImageBlock(url="https://example.com/image.png")
    ]
)
```

### 3. Pipeline 抽象

```python
from agentscope.pipeline import SequentialPipeline

# 创建流水线
pipeline = SequentialPipeline([
    agent1,
    agent2,
    agent3
])

# 执行
result = pipeline(user_input)
```

## 使用示例

### 1. 简单对话 Agent

```python
from agentscope.agent import DialogAgent
from agentscope.memory import TemporaryMemory
from agentscope.model import OpenAIChatWrapper

# 创建模型
model = OpenAIChatWrapper(
    model_name="gpt-4",
    api_key="your-api-key"
)

# 创建 Agent
agent = DialogAgent(
    name="assistant",
    sys_prompt="You are a helpful assistant.",
    model_config_name="gpt-4",
    memory=TemporaryMemory()
)

# 对话
response = agent.reply(Msg(name="user", content="Hello!"))
print(response.content)
```

### 2. 多智能体协作

```python
from agentscope.agent import ReActAgent
from agentscope.pipeline import SequentialPipeline

# 创建多个 Agent
planner = ReActAgent(name="planner", ...)
researcher = ReActAgent(name="researcher", ...)
writer = ReActAgent(name="writer", ...)

# 创建流水线
pipeline = SequentialPipeline([planner, researcher, writer])

# 执行
result = pipeline(Msg(name="user", content="Write a research report"))
```

### 3. RPC 分布式 Agent

```python
from agentscope.rpc import RpcAgent

# 创建 RPC Agent（远程）
remote_agent = RpcAgent(
    name="remote_agent",
    host="192.168.1.100",
    port=8080
)

# 调用远程 Agent
response = remote_agent.reply(Msg(name="user", content="Process this"))
```
