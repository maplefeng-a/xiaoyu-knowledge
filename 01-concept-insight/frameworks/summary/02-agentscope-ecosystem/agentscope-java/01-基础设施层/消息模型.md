
# Msg 消息模型

> agentscope-java 基础设施层 | 核心通信单元

## 一句话

不可变的多模态消息抽象，Agent 间通信的基本单元，支持文本、图片、音频、工具调用等 7 种内容类型。

---

## 核心设计

### 设计原则
- **不可变性**: 所有字段 final，线程安全
- **Builder 模式**: 流式构建消息
- **多态内容**: `List<ContentBlock>` 支持多种类型
- **类型安全**: Sealed class + 泛型方法

### Msg 结构

```java
Msg {
  id: String                        // UUID 自动生成
  name: String                      // 可选名称
  role: MsgRole                     // USER/ASSISTANT/SYSTEM/TOOL
  content: List<ContentBlock>       // 内容块列表（不可变）
  metadata: Map<String, Object>     // 元数据
  timestamp: String                 // 时间戳
}
```

---

## ContentBlock 类型

### 继承关系（Sealed Class）

```
ContentBlock (sealed)
├── TextBlock          // 文本
├── ThinkingBlock      // 思考过程
├── ImageBlock         // 图片
├── AudioBlock         // 音频
├── VideoBlock         // 视频
├── ToolUseBlock       // 工具调用请求
└── ToolResultBlock    // 工具执行结果
```

### 类型详解

| 类型 | 用途 | 关键字段 | JSON type |
|------|------|----------|-----------|
| **TextBlock** | 文本内容 | `text` | `text` |
| **ThinkingBlock** | Agent 思考过程 | `thinking`, `metadata` | `thinking` |
| **ImageBlock** | 图片 | `source` (URL/Base64) | `image` |
| **AudioBlock** | 音频 | `source` | `audio` |
| **VideoBlock** | 视频 | `source` | `video` |
| **ToolUseBlock** | 工具调用请求 | `id`, `name`, `input` | `tool_use` |
| **ToolResultBlock** | 工具执行结果 | `id`, `name`, `output` | `tool_result` |

---

## MsgRole 枚举

| 角色 | 说明 | 使用场景 |
|------|------|----------|
| `USER` | 用户消息 | 用户输入、外部请求 |
| `ASSISTANT` | Agent 响应 | Agent 输出 |
| `SYSTEM` | 系统指令 | System Prompt |
| `TOOL` | 工具结果 | 工具返回值 |

## 关键方法

### 内容提取

```java
// 获取纯文本（拼接所有 TextBlock）
String text = msg.getTextContent();

// 类型安全获取
List<TextBlock> texts = msg.getContentBlocks(TextBlock.class);
TextBlock first = msg.getFirstContentBlock(TextBlock.class);

// 检查是否存在
boolean hasToolUse = msg.hasContentBlocks(ToolUseBlock.class);
```

### 结构化输出

```java
// 检查并获取结构化数据
if (msg.hasStructuredData()) {
    TaskPlan plan = msg.getStructuredData(TaskPlan.class);
    Map<String, Object> map = msg.getStructuredData(false);
}
```

### 元数据

```java
// Token 使用统计
ChatUsage usage = msg.getChatUsage();

// 生成原因
GenerateReason reason = msg.getGenerateReason();
// MODEL_STOP / TOOL_SUSPENDED / REASONING_STOP_REQUESTED / ...
```

## 使用示例

### 创建消息

```java
// 简单文本消息
Msg userMsg = Msg.builder()
    .role(MsgRole.USER)
    .textContent("你好")
    .build();

// 多内容块消息
Msg msg = Msg.builder()
    .role(MsgRole.USER)
    .content(
        TextBlock.builder().text("这张图片是什么？").build(),
        ImageBlock.builder().url("https://example.com/image.png").build()
    )
    .name("user")
    .build();
```

### 处理响应

```java
Msg response = agent.call(userMsg).block();

// 提取文本
String text = response.getTextContent();

// 提取思考过程
List<ThinkingBlock> thinking = response.getContentBlocks(ThinkingBlock.class);

// 提取工具调用
List<ToolUseBlock> toolCalls = response.getContentBlocks(ToolUseBlock.class);
for (ToolUseBlock tool : toolCalls) {
    String toolName = tool.getName();
    Map<String, Object> input = tool.getInput();
}
```
