## 📈 关键收获

### 1. MsgHub 核心机制

```java
// 自动广播原理
private void resetSubscribers() {
    for (AgentBase agent : participants) {
        List<AgentBase> others = participants.stream()
            .filter(a -> !a.equals(agent))
            .collect(Collectors.toList());
        agent.resetSubscribers(name, others);
    }
}
```

**关键点**：
- 每个 Agent 订阅其他所有 Agent
- 使用 `CopyOnWriteArrayList` 保证线程安全
- `enter()/exit()` 管理生命周期

### 2. Pipeline 两种模式

| Pipeline | 执行方式 | 返回类型 | 适用场景 |
|----------|---------|---------|---------|
| SequentialPipeline | 顺序链式 | `Mono<Msg>` | 研究→分析→撰写 |
| FanoutPipeline | 并行/顺序 | `Mono<List<Msg>>` | 多语言翻译 |

### 3. ReActAgent 核心循环

```
用户输入 → Reasoning → 有工具? → Acting → Reasoning → ... → Summarizing
```

**关键方法**：
- `doCall()` - 入口，处理待处理工具
- `reasoning()` - LLM 推理，流式累积
- `acting()` - 执行待处理工具
- `summarizing()` - maxIters 时生成总结

### 4. ReActAgent Hook 事件

| 阶段 | Pre Hook | Post Hook | Chunk Hook |
|------|----------|-----------|------------|
| Reasoning | PreReasoningEvent | PostReasoningEvent | ReasoningChunkEvent |
| Acting | PreActingEvent | PostActingEvent | ActingChunkEvent |
| Summary | PreSummaryEvent | PostSummaryEvent | SummaryChunkEvent |
